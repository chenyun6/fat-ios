# 隐私政策草案

更新时间：2026-03-22

《今天你胖了吗？》非常重视用户个人信息和隐私保护。本政策用于说明我们如何收集、使用、共享、保存和删除你的个人信息，以及你享有的相关权利。

## 1. 我们收集哪些信息

为了向你提供服务，我们可能收集以下信息：

- 账号与登录信息：手机号、验证码登录产生的用户编号、访问令牌、刷新令牌
- 业务数据：你每日提交的“胖了 / 没胖”记录、记录日期、记录时间
- 必要的日志信息：接口请求时间、IP、设备基础信息、错误日志，用于排查故障和保障服务安全

我们当前不会主动向你请求通讯录、相册、定位、麦克风等与核心功能无关的权限数据。

## 2. 我们如何使用这些信息

我们收集和使用你的信息，主要用于：

- 完成手机号验证码登录和账号识别
- 维持登录状态并保障账号安全
- 保存、展示和同步你的每日记录
- 防止重复打卡、接口滥用与服务异常
- 进行必要的运维、风控、故障分析与问题排查

## 3. 我们是否共享你的信息

- 除法律法规另有要求外，我们不会出售你的个人信息。
- 为完成核心功能所必需，我们可能与短信服务提供商、云服务器或基础设施服务商共享必要信息，例如手机号、验证码发送请求或访问日志。
- 如接入第三方服务商，我们会要求其按照不低于本政策的标准保护你的个人信息。

## 4. 数据保存与删除

- 我们会在实现产品功能和满足法律法规要求所必需的期限内保存你的个人信息。
- 你可通过应用内后续提供的账号删除能力，或发送邮件至 171811603@qq.com 申请删除账号与相关数据。
- 收到删除申请后，我们会在合理期限内处理，并在法律法规要求的保留期限届满后删除或匿名化处理相关数据。

发布前请确认：如果你的 App 已支持账号创建，则建议在 App 内提供明确的账号删除入口，并保证本条描述与实际能力一致。

## 5. 你的权利

你有权依法访问、更正、删除你的个人信息，并可在适用范围内撤回授权同意。你也可以通过隐私邮箱联系我们行使相关权利。

## 6. 未成年人保护

如果你是未成年人，请在监护人阅读并同意本政策后使用本服务。如我们发现未经监护人同意收集了未成年人信息，将尽快删除相关信息或停止处理。

## 7. 政策更新

当产品功能、服务方式、法律法规或监管要求发生变化时，我们可能更新本政策。更新后的内容将通过应用内、官网页面或其他合理方式公布。
