### 步骤 2: 配置 Info.plist

#### 方法 A：使用 Xcode 界面（推荐）

1. **选择项目和 Target**
   - 点击左侧导航栏最顶部的 "fat" 项目图标（蓝色图标）
   - 在中间区域的 TARGETS 列表中，点击 "fat"

2. **打开 Build Settings**
   - 点击顶部的 "Build Settings" 标签
   - 在搜索框中输入 "info"

3. **修改 Info.plist 配置**
   查找以下两个设置：

   **设置 1：** `Generate Info.plist File`
   - 找到 "Generate Info.plist File"
   - 将值改为 **`NO`**

   **设置 2：** `Info.plist File`
   - 找到 "Info.plist File"
   - 双击值的位置，输入：**`fat/Info.plist`**

4. **保存**
   - Command + S 保存

---

#### 方法 B：手动编辑项目文件（备选）

如果方法 A 不生效，使用此方法：

1. **关闭 Xcode**

2. **编辑项目文件**
   ```bash
   open -a TextEdit /Users/huyingjun/handsome/fat-ios/fat.xcodeproj/project.pbxproj
   ```

3. **查找并修改**

   使用 Command + F 搜索：`GENERATE_INFOPLIST_FILE`

   找到类似这样的行：
   ```
   GENERATE_INFOPLIST_FILE = YES;
   ```

   改为：
   ```
   GENERATE_INFOPLIST_FILE = NO;
   ```

4. **添加 Info.plist 路径**

   在同一个配置块中，找到或添加：
   ```
   INFOPLIST_FILE = fat/Info.plist;
   ```

   完整示例：
   ```
   buildSettings = {
       ...
       GENERATE_INFOPLIST_FILE = NO;
       INFOPLIST_FILE = fat/Info.plist;
       ...
   };
   ```

5. **保存并重新打开 Xcode**

---

### 步骤 3: 清理并重新编译

1. **清理构建缓存**
   - 在 Xcode 菜单栏：Product → Clean Build Folder
   - 或按快捷键：**Command + Shift + K**

2. **删除 Derived Data（可选但推荐）**
   ```bash
   rm -rf ~/Library/Developer/Xcode/DerivedData/fat-*
   ```

3. **重新编译**
   - Command + B（Build）

4. **运行项目**
   - Command + R（Run）

## 🔧 常见问题排查

### 问题 1: Xcode 中找不到 Info.plist 文件

**解决方案：**
1. 在 Xcode 左侧文件导航中，右键点击 "fat" 文件夹
2. 选择 "Add Files to 'fat'..."
3. 找到并选择 `fat/Info.plist`
4. 确保勾选 "Copy items if needed" 和 "fat" target
5. 点击 "Add"

---

### 问题 2: 修改后仍然报 ATS 错误

**检查清单：**

- [ ] 确认 `GENERATE_INFOPLIST_FILE = NO`
- [ ] 确认 `INFOPLIST_FILE = fat/Info.plist`
- [ ] 执行了 Clean Build Folder
- [ ] 删除了 Derived Data
- [ ] 完全退出并重启了 Xcode
- [ ] 重新运行了应用

**强制清理：**
```bash
# 1. 关闭 Xcode

# 2. 删除所有缓存
rm -rf ~/Library/Developer/Xcode/DerivedData
rm -rf ~/Library/Caches/com.apple.dt.Xcode

# 3. 删除项目缓存
cd /Users/huyingjun/handsome/fat-ios
rm -rf .build
rm -rf fat.xcodeproj/xcuserdata
rm -rf fat.xcodeproj/project.xcworkspace/xcuserdata

# 4. 重新打开 Xcode
open fat.xcodeproj
```

---

### 问题 3: 网络请求超时

**检查后端服务：**

```bash
# 测试后端是否可访问
curl -v http://11.192.195.121:8080/web/weight/send-code

# 或者
ping 11.192.195.121
```

**确认：**
- [ ] 后端服务已启动
- [ ] IP 地址正确：`11.192.195.121`
- [ ] 端口正确：`8080`
- [ ] Mac 和后端在同一网络

---

### 问题 4: Info.plist 不生效

**验证 Info.plist 是否被使用：**

1. 运行应用
2. 在 Xcode 菜单：Window → Devices and Simulators
3. 选择当前运行的设备/模拟器
4. 点击 "Installed Apps"
5. 找到 "fat" 应用，点击齿轮图标
6. 选择 "Show Container"
7. 查看 Info.plist 内容，确认是否包含 ATS 配置

**或者使用命令行：**
```bash
# 构建后，查看 Info.plist
cat ~/Library/Developer/Xcode/DerivedData/fat-*/Build/Products/Debug-iphonesimulator/fat.app/Info.plist
```
